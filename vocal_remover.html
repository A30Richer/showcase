AI Vocal & Music Separator

เครื่องมือแยกเสียงร้อง (Vocals) และเสียงดนตรีแบ็กกิ้งแทร็ก (Instrumental) ด้วยสถาปัตยกรรม Meta Demucs v4 (State-of-the-Art) รันฟรีประมวลผลบนคลาวด์ผ่าน GitHub Actions

ขั้นตอนการรันแยกเสียงบน GitHub Actions (ฟรี 100%)
1

วางลิงก์เพลง หรือ YouTube ที่ต้องการแยกเสียง

กรอกลิงก์ YouTube (เพลง/Shorts) หรือลิงก์ตรงของไฟล์เสียงในช่องด้านล่าง แล้วกดคัดลอก

2

กดปุ่มเปิดหน้า GitHub Actions Workflow

ระบบจะพาคุณไปยังหน้า **GitHub Actions (`vocal_remover.yml`)** ปลอดภัย 100% ไม่ต้องกรอก Token รหัสผ่านใดๆ

3

กดปุ่ม "Run workflow" และรอรับไฟล์ ZIP ผลลัพธ์

วางลิงก์เพลงแล้วกดรัน เมื่อระบบประมวลผลเสร็จ (ประมาณ 1-2 นาที) สามารถกดดาวน์โหลดไฟล์ **`separated-audio-result`** ได้ทันที!

🚀 เปิดหน้า GitHub Actions เพื่อสั่งรันแยกเสียง
ตัวอย่างคุณภาพเสียงที่แยกได้จาก Meta Demucs AI

ทดลองฟังเปรียบเทียบคุณภาพไฟล์เสียงที่แยกเสียงร้องและดนตรีออกจากกันด้วยโมเดล Demucs v4:

🎤 Isolated Vocals (เสียงร้องเดี่ยว)
ตัดเสียงดนตรีออกทั้งหมด เสียงร้องคมชัด ไร้เสียงรบกวน
🎸 Instrumental Track (เสียงดนตรีคาราโอเกะ)
ดนตรีแบ็กกิ้งแทร็กสะอาด สำหรับนำไปร้องเพลงหรือมิกซ์ใหม่
