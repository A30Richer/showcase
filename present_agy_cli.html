AUTHORITATIVE COMMAND REFERENCE

AGY CLI & Slash Commands Complete Guide

คู่มือสรุปคำสั่งทั้งหมดอย่างครอบคลุมถ้วนหน้า อ้างอิงจากคำสั่ง agy --help และ TUI Slash Commands พร้อมระบบจำลองสถานการณ์ตัวอย่าง (Scenario Dropdown)

1. คำสั่งย่อยของ AGY CLI (Subcommands)
agy agent / agents ดูรายชื่อ Agent

แสดงรายการ Agents ทั้งหมดที่เปิดใช้งานได้ในระบบ

agy changelog บันทึกการอัปเดต

แสดงรายการอัปเดตเวอร์ชันและฟีเจอร์ใหม่ล่าสุดของ Antigravity CLI

agy help [command] ช่วยเหลือ

แสดงเมนูช่วยเหลือและคู่มือการใช้งานคำสั่งย่อยต่างๆ

agy install ติดตั้ง Environment

ตั้งค่า Environment PATHS และ Shell Completion สำหรับการใช้งานระบบ CLI

agy models เช็กโมเดล AI

แสดงรายชื่อโมเดล AI ทั้งหมดที่มีให้เลือกรันในระบบ (เช่น Gemini 3.6 Flash, Pro, ฯลฯ)

agy plugin / plugins จัดการพัคอิน

บริหารจัดการส่วนเสริม Plugins (install, uninstall, list, enable, disable)

จำลองสถานการณ์: agy plugin
📌 สถานการณ์
คุณต้องการติดตั้งหรือดูปลั๊กอินส่วนเสริมเพื่อขยายความสามารถของ CLI
💬 คำสั่งที่ใช้
agy plugin list
agy plugin enable my-extension
agy update อัปเดตเวอร์ชัน

ตรวจสอบและอัปเดต Antigravity CLI ให้เป็นเวอร์ชันล่าสุดโดยอัตโนมัติ

2. พารามิเตอร์และตัวเลือก CLI Flags (`agy --flag`)
--add-dir <path> เพิ่มไดเรกทอรีทำงาน

เพิ่มโฟลเดอร์ภายนอกเข้าเป็นส่วนหนึ่งของ Workspace การทำงาน (สามารถพิมพ์ซ้ำหลายโฟลเดอร์ได้)

-c | --continue สานต่อแชทล่าสุด

เปิดใช้งาน CLI และสานต่อบทสนทนาจาก Session ล่าสุดทันที

--conversation <id> เจาะจง Session ID

เปิดโหลดบทสนทนาเก่าเฉพาะเจาะจงตาม Conversation ID

--dangerously-skip-permissions ข้ามการขออนุมัติ

อนุมัติคำขอสิทธิ์การเขียนไฟล์และรันคำสั่งทั้งหมดโดยอัตโนมัติ ไม่ต้องเด้งถามผู้ใช้

จำลองสถานการณ์: --dangerously-skip-permissions
📌 สถานการณ์
คุณต้องการรันสคริปต์อัตโนมัติใน CI/CD ที่มั่นใจความปลอดภัยแล้ว และไม่ต้องการให้กระบวนการหยุดรอการกดยืนยันจากมนุษย์
💬 คำสั่งที่ใช้
agy run "รันเทสต์ทั้งหมดและแก้ลิ้นท์" --dangerously-skip-permissions
--effort <low | medium | high> ปรับระดับความคิด

กำหนดระดับความลึกในการประมวลผลให้ AI (Low = เร็วประหยัด, High = คิดวิเคราะห์ซับซ้อน)

-p | --print | --prompt <text> รันคำสั่งสั้นจบในตัว

สั่งรันงานสั้นๆ แบบ Non-Interactive แล้วพิมพ์ผลลัพธ์ออกหน้า Terminal ทันทีแล้วจบ

จำลองสถานการณ์: -p / --print
📌 สถานการณ์
คุณต้องการส่ง Prompt สั้นๆ จาก Bash script เพื่อตรวจคำผิดในไฟล์
💬 คำสั่งที่ใช้
agy -p "ตรวจทานคำผิดใน README.md"
--mode <accept-edits | plan> กำหนดโหมดการทำงาน

ตั้งค่าโหมดเริ่มต้นของ Agent ใน Session เช่น โหมดเขียนไฟล์ทันที หรือโหมดเน้นวางแผนก่อน

--sandbox โหมดปลอดภัย Sandbox

เปิดใช้งานโหมดจำกัดสิทธิ์ความปลอดภัย ห้ามรันคำสั่งอันตรายนอกพื้นที่ที่กำหนด

3. คำสั่ง Slash Commands ในหน้าแชท TUI (`/`)
/goal [คำอธิบายเป้าหมาย] โหมดลุยงานยาวข้ามคืน

สั่ง AI ลุยเป้าหมายใหญ่ต่อเนื่อง แก้โค้ด รันเทสต์ วนลูปไม่หยุดจนกว่าเป้าหมายจะสำเร็จ 100%

จำลองสถานการณ์: /goal
📌 สถานการณ์
สั่งให้ทำ Refactor และเพิ่ม Unit Test ให้ครบทั้ง 50 ไฟล์ในโปรเจกต์ ปล่อยให้ระบบรันข้ามคืน
💬 คำสั่งที่ใช้
/goal เขียน Unit Test ให้ครบทุกไฟล์ใน tests/ และห้ามหยุดจนกว่าจะผ่าน 100%
/schedule [ระยะเวลา / เงื่อนไข] ตั้งเวลา / Cron Job

ตั้งเวลานาฬิกาปลุก หรือตั้ง Cron Schedule ให้ AI กลับมาตรวจเช็กงานหรือแจ้งเตือนตามรอบเวลา

จำลองสถานการณ์: /schedule
📌 สถานการณ์
เพิ่งสั่ง Deploy หน้าเว็บ แล้วอยากให้ AI กลับมาตรวจเช็ก HTTP Status Code ในอีก 5 นาที
💬 คำสั่งที่ใช้
/schedule ในอีก 5 นาที ให้ยิงเช็ก URL ว่าขึ้น 200 OK หรือยัง
/plan [เป้าหมาย] วางแผนพิมพ์เขียวก่อนแตะโค้ด

บังคับให้ AI สรุปแผนงานเป็นข้อๆ (Blueprint Plan) ให้มนุษย์กดอนุมตรก่อนลงมือแก้ไฟล์จริง

/grill-me สัมภาษณ์ปรับความเข้าใจ

ให้ AI ยิงคำถามสัมภาษณ์คุณยิบๆ (Interview Mode) เพื่อเก็บความต้องการและเคลียร์ข้อสงสัยก่อนเริ่มงาน

/teamwork-preview แตกงานให้ทีม Subagents

สปอว์นทีมงาน Subagents หลายตัวมาช่วยกันรันงานขนานกันไปพร้อมๆ กัน

/learn บันทึกความจำ / กฎระเบียบ

บันทึกสไตล์การโค้ดหรือคำแนะนำของคุณลงโฟลเดอร์ .gemini/rules/ ให้ AI จำไว้ใช้ตลอดไป

/compact ย่อประวัติแชท

สรุปรวมประวัติบทสนทนาที่ยาวเกินไปให้สั้นลง เพื่อประหยัด Context Window ในการทำงาน

/copy คัดลอกคำตอบ

คัดลอกโค้ดหรือข้อความตอบกลับล่าสุดของ AI เข้าสู่ Clipboard ของเครื่องทันที

/clear | /help | /exit คำสั่งควบคุมระบบ

/clear: ล้างประวัติแชทเริ่มต้นใหม่ | /help: ดูเมนูคำสั่งช่วยเหลือ | /exit: ปิดและออกจากโปรแกรม
